Sistema Reportes - CRUD

Java/Mysql Workbench/NetBeans

Fecha: 06/04/2020

Puedes verlo en: GitHub.com

Una aplicación de consola, que permite guardar reportes de técnicos e información sobre algunos usuarios. Al desarrollar esta aplicación aprendí a conectar una base de datos MySQL con Java mediante un controlador. Aprendí a trabajar con POO, realicé un menú donde el usuario interactúa y manipula la información, dicha información esta almacenada en una BD. Con MySQL Workbench diseñe la base de datos y mediante SQL realice las consultas básicas para Listar, Eliminar, Actualizar e Insertar datos.
