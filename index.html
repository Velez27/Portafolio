App Web - Rick And Morty

JavaScript/HTML/CSS

Fecha: 10-05-2020

Puedes verlo en: GitHub.com

Aplicación Web, permite visualizar información de los personajes de la serie animada Rick and Morty. Utilice HTML para la estructura y CSS para dar estilos y aplicar los colores de la aplicación. Al desarrollar esta aplicación aprendí a consumir recursos de una API con JavaScript, realicé la lógica para solo consumir solo la información que requería de la API (imagen, nombre, origen, etc.), también se crearon funciones para manipular el DOM y mediante un botón el usuario pueda ver diferentes personajes de manera aleatoria.

Sistema Reportes - CRUD

Java/Mysql Workbench/NetBeans

Fecha: 06/04/2020

Puedes verlo en: GitHub.com

Una aplicación de consola, permite guardar reportes de soporte e información sobre técnicos y administrativos relacionados a dichos reportes. Al desarrollar esta aplicación aprendí a conectar una base de datos MySQL con Java mediante un controlador. Aprendí a trabajar con POO, ya que Java está orientado a este paradigma, realicé la lógica para crear un menú donde el usuario interactúa y realiza las operaciones y el manejo de la información en la Base de datos. Con MySQL Workbench se diseñó la base de datos y mediante SQL realice las consultas básicas para Listar, Eliminar, Actualizar e Insertar datos.
